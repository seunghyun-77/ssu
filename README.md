# PPSP C simulator

Final-paper timing is authoritative: candidates/public parameters are revealed
first; `W` and realized external `R`/`q` are revealed at the win-confirm month
regardless of proposal; the information state is updated; then decisions occur.
Contractual obligation still requires a prior proposal and `W=1`.

The CSV package supplies base profiles instead of explicit scenario-indexed
`R`/`q` tables, so the implementation maps them to the base profiles plus the
scenario market/resource multipliers. P0-P7 are core paper policies; P8-P9 are
sensitivity policies.

Replication-level potential wins are regenerated with policy-independent common
random numbers because the supplied win table has no replication dimension.
See `METHODOLOGY.md` for the correction, aggregation rule, and audit trail.

The reference experiment contains 14,000 runs (10 policies x 70 scenarios x 20
replications), uses a 5% annual discount rate and a 1.0 internal-ratio penalty,
and writes its audited thesis tables to `experiment_output_base`.

For exact build, test, full-run, statistical-table, and sensitivity commands,
see `REPRODUCIBILITY.md`. The principal Korean results narrative is
`experiment_output_base/THESIS_RESULTS_KO.md`.

Seven dependency-free SVG dissertation figures and their Korean interpretation
are generated by `tools/generate_thesis_charts.py`. See
`experiment_output_base/THESIS_VISUAL_INTERPRETATION_KO.md`.

The extended 14,000-run replication in `experiment_output_project` adds 2.66
million candidate-project decision traces. Its main interpretation is in
`experiment_output_project/PROJECT_LEVEL_RESULTS_KO.md`; the trace audit is in
`experiment_output_project/PROJECT_DECISION_AUDIT.json`.

Failure decomposition, practical significance, and held-out policy-selection
validation are documented in
`experiment_output_project/DISSERTATION_ROBUSTNESS_RESULTS_KO.md`.

Axis meanings, units, legends, overlap corrections, and result interpretations
for all 14 figures are documented in
`experiment_output_project/FIGURE_AXIS_AND_INTERPRETATION_KO.md`.
