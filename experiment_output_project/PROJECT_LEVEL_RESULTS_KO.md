# 프로젝트 수준 정책효과 분석

## 분석 범위

프로젝트 의사결정 추적 기능을 추가한 뒤 기준 모수(연 할인율 5%, 내부매출
비율 페널티 1.0)로 14,000회를 다시 실행하였다. 고정 프로젝트 10개를
제외한 내부 후보 80개와 외부 후보 110개에 대해 실행별 관측월, 제안월,
잠재 수주결과, 착수월, 할인 총수익 및 자원수요를 기록하였다.

총 2,660,000개 기록은 `14,000회 x 190개 후보 프로젝트`로 구성된다.
재실행의 기존 핵심 결과는 이전 기준 실험과 완전히 동일하였다. 비교한
항목은 목적함수, 수익, 비용, 수주율, 활용률, 유휴, 부족, 내부매출 비율
위반 및 실패판정이다.

## P8이 P0보다 높은 성과를 낸 프로젝트 경로

P8은 단순히 모든 프로젝트를 더 많이 착수한 정책이 아니었다. 정책별
프로젝트 구성이 다음과 같이 이동하였다.

- 외부 후보 평균 제안률: P0 0.8375, P8 0.9043
- 외부 후보 평균 착수율: P0 0.4153, P8 0.4415
- 내부 후보 평균 착수율: P0 0.3865, P8 0.3704
- 외부 프로젝트 할인 총수익 합계: P0 115,528.1, P8 120,903.0
- 내부 프로젝트 할인 총수익 합계: P0 39,994.7, P8 37,524.5
- 전체 후보 프로젝트 할인 총수익 합계: P0 155,522.8, P8 158,427.5

즉, P8은 내부 후보를 일률적으로 확대하지 않았다. 외부 프로젝트 제안과
착수 기회를 더 적극적으로 확보하고 일부 내부 프로젝트를 줄이는
포트폴리오 재구성을 통해 후보 프로젝트 할인 총수익을 약 2,904.7
증가시켰다. 최종 목적함수 개선치 1,993.5가 이보다 작은 것은 인력·외주·
유휴·부족·제안·지연비용과 내부매출 비율 페널티가 함께 반영되기 때문이다.
공유비용은 개별 프로젝트에 임의 배분하지 않았으므로 이 차이를 프로젝트별
순이익 기여로 해석해서는 안 된다.

## 프로젝트별 이질성

P8과 P0의 착수율 차이가 가장 큰 프로젝트는 다음과 같다.

- I044: -0.2364
- I004: +0.2150
- I058: +0.1814
- I069: -0.1686
- I041: -0.1579
- E099: +0.1486
- E006: +0.1429
- E106: +0.1321

이 결과는 정책효과가 모든 프로젝트에 동일하게 적용되지 않고, 프로젝트의
기대가치·자원부담·관측시점·수주확률과 정책 가중치의 결합에 따라 선택적으로
나타남을 보여준다. 따라서 P8의 효과를 단순한 “인력 증가 효과”가 아니라
“선제적 자원확보를 통한 프로젝트 포트폴리오 구성 변화”로 해석할 수 있다.

## 시나리오별 해석

P8의 외부 프로젝트 착수율은 분석한 모든 시나리오 그룹에서 P0보다
높았다. 반대로 내부 프로젝트 착수율은 모든 그룹에서 P0보다 낮았다.
LOW_WIN에서는 외부 착수율 개선 폭이 상대적으로 작고 내부 프로젝트 감소가
나타나 P8의 정책 전체 순위가 2위로 내려간다. 이는 낮은 수주환경에서는
외부 제안 확대의 기대편익이 줄어든다는 정책 메커니즘과 일치한다.

## 타당성 감사

제안·수주 없이 외부 프로젝트 착수, 착수 없이 수익·자원 기여 발생,

해시는 `PROJECT_DECISION_AUDIT.json`에 기록하였다.
